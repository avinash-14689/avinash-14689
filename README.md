## 🚀 Featured Projects

### 🤖 [AEVI — Personal AI Assistant](https://github.com/avinash-14689/AEVI)
> A full-stack personal AI assistant built with Flask + Groq LLM

- 🏠 Smart home controls — lights, fan, AC, door lock
- 🎵 Spotify playback control
- 📅 Google Calendar integration
- 🌤️ Live weather updates
- 🧠 Powered by **Groq API (Llama 3.3 70B)**
- 🛠️ Built with Flask, Python, HTML/CSS/JS

---

### 🖐️ [Hand Gesture Controller](https://github.com/avinash-14689/Hand_Gestures)
> Control your computer using just hand gestures — no touch needed

- 🔄 Swipe to switch browser tabs
- 📜 Scroll pages with hand movement
- ❌ Close apps with cross gesture
- 🛠️ Built with **MediaPipe + OpenCV + PyAutoGUI**

- 🗣️ Google Speech Recognition
